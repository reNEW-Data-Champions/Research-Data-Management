# Accessibility

Accessibility ensures that data and metadata can be easily accessed and retrieved by humans and machines.

* A1. Make the data and metadata accessible through a standardized communications protocol, such as an API or web service, with proper authentication and authorization mechanisms.
* A2. Provide clear instructions and documentation on accessing and using the data, including any required software, tools, or libraries.
* A3. Ensure that data access complies with relevant data protection and privacy regulations and that any necessary consent or permissions are obtained.
* A4. Encourage long-term data preservation by storing data in a sustainable and trusted repository and providing access even after the project's completion.
